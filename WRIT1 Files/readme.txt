# CIS5004 WRIT1: Data and Knowledge Management – SmartShop Ltd.

## Overview
This repository contains all code and data for the analytical report on SmartShop Ltd., covering transaction management, concurrency control, Power BI visualisations, and a NoSQL implementation in MongoDB.

## Files Included
- `schema.sql` – MySQL table creation
- `data.sql` – Sample data
- `queries.sql` – Analytical SQL queries
- `transactions.sql` – Transaction examples
- `concurrency_demo.sql` – Concurrency control demo
- `SmartShop_Dashboard.pbix` – Power BI dashboard
- `customers.json` – MongoDB customer data
- `products.json` – MongoDB products with embedded reviews

## Setup Instructions
1. **MySQL**: Run `schema.sql` then `data.sql` in MySQL Workbench.
2. **Concurrency Demo**: Open two query tabs, follow the instructions in `concurrency_demo.sql`.
3. **MongoDB**: Import `customers.json` and `products.json` into a database named `smartshop_nosql`.
4. **Power BI**: Open the `.pbix` file to view the dashboard.
