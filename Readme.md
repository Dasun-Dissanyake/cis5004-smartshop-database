# SmartShop Ltd. – Database Design and Implementation

## 📋 Project Overview
This repository contains the complete database implementation for SmartShop Ltd., a mid-sized retail company with physical stores and an online platform. The project fulfills the requirements for CIS5004 Data and Knowledge Management assessment.

## 🗄️ Repository Structure
- `schema.sql` – MySQL table creation scripts
- `data.sql` – Sample data insertion
- `queries.sql` – Analytical SQL queries (top products, monthly sales, etc.)
- `transactions.sql` – Transaction examples (commit/rollback)
- `SmartShop_Dashboard.pbix` – Power BI dashboard
- `reviews.json` – Sample customer reviews for MongoDB
- `logs.json` – Sample website logs for MongoDB

## 🛠️ Technologies Used
- **MySQL** – Relational database management
- **Power BI** – Data visualization and business intelligence
- **MongoDB** – NoSQL database for unstructured data

## 🚀 Key Features
- Normalized relational schema with customers, branches, products, orders
- ACID-compliant transactions with commit/rollback demonstrations
- Analytical queries for business insights
- Interactive Power BI dashboard
- NoSQL implementation for customer reviews and website logs
